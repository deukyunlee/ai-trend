## Harness & Orchestration

- **[The Interplay of Harness Design and Post-Training in LLM Agents](http://arxiv.org/abs/2606.25447v1)**
  하네스 구성(도구 노출·관찰 정보)이 post-training 결과에 직접 영향. 하네스를 학습 변수로 다뤄야 한다는 주장.

- **[ActPlane: Programmable OS-Level Policy Enforcement for Agent Harnesses](http://arxiv.org/abs/2606.25189v1)**
  자연어 정책을 OS 수준 시스템 액션으로 브리징하는 런타임. "커밋 전 테스트 실행" 같은 정책을 선언적으로 집행.

- **[A Process Harness for Uplifting Legacy Workflows to Agentic BPM](http://arxiv.org/abs/2606.27188v1)**
  기존 워크플로우 엔진을 교체하지 않고 agentic 레이어를 감싸는 "process harness" 패턴.

- **[NOVA: Verification-Aware Agent Harness for Architecture Evolution](http://arxiv.org/abs/2606.27243v1)**
  산업용 추천 시스템 아키텍처 진화를 위한 검증 인식 하네스.

- **[AOHP: OS-Level Agent Harness for Personalized, Efficient and Secure Interaction](http://arxiv.org/abs/2606.23449v1)**
  AI 에이전트를 위한 OS 수준 오픈소스 하네스. 기존 앱 중심 OS의 한계를 극복.

## Loop Engineering

- **[Semantic Early-Stopping for Iterative LLM Agent Loops](http://arxiv.org/abs/2606.27009v1)**
  임베딩 cosine distance로 루프 종료를 결정하는 semantic kill-switch. 고정 cap 대신 입력 난이도에 따라 동적으로 반복 횟수 조절.

- **[From Prompts to Loops: Building Autonomous Coding Agents](https://news.ycombinator.com/item?id=x)** _(HN: 4)_
  단순 프롬프트에서 자율 루프 기반 코딩 에이전트로 전환하는 실전 가이드.

- **[Grounded Iterative Language Planning: How Parameterized World Models Reduce Hallucination Propagation in LLM Agents](http://arxiv.org/abs/2606.x)**
  파라미터화된 월드 모델로 반복 계획 중 환각 누적을 억제하는 접근법.

- **[Don't Blindly Trust It: How Unreliable Feedback Breaks Tool-Using LLM Agents](http://arxiv.org/abs/2606.21409v1)**
  루프 내 신뢰할 수 없는 피드백이 오히려 피드백 없는 것보다 성능을 저하시킴을 실증.

- **[Bridging Creative Intent and Visual Quality: Creator-Driven Recurrent Video Generation with Agentic Feedback Loops](http://arxiv.org/abs/2606.x)**
  크리에이터 의도와 시각 품질 간극을 agentic feedback loop으로 좁히는 영상 생성 시스템.

## Evaluation & Benchmarks
